Kontaktliste

Übung folgender JavaScript Features:

  • Auslesen von Formulardaten
  • Sammeln von Daten in Form von Objektliteralen
  • Umwandeln von JSON Daten zwischen Objekt und String
  • Speichern in und Auslesen aus localStorage
  • Daten als Tabelle ausgeben

Ideen zur Weiterentwicklung

  • Kontaktliste um ein Foto erweitern, das per Drag & Drop hinzugefügt wird.
  • Kontakte erfassen und Kontakte anzeigen auf verschiedenen Seiten durchführen.
  • Kontaktliste als HTML Datentabelle anzeigen.
  • Lösch- und Editierfunktionen für Kontakte ergänzen.
  • Sicherheitsfunktionen für die Säuberung und Validierung von Benutzereingaben ergänzen.
